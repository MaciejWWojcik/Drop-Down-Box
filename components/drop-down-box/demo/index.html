Basic drop-down-box

You can pre-select drop-down-box value using "default" property

You can disable empty drop-down-box as well as a filled in one using "is-disable" parameter

drop-down-box can also take "is-wrong" parameter

"is-active" is standard behaviour when user clicks on an item, but it can be a parameter too

drop-down-box is responsive. By default it is 218px by 42px, but developer can change it to whatever he wants

drop-down-box component has its own validation. By default it is set to true.

This example shows behaviour of a component when user wants to get an item using component.getCurrentItem(). Test it with blank and filled box.

Another example shows behaviour of drop-down-box when user decides to dynamically disable/enable parameter
